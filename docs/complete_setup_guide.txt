# Complete DaySave Setup Guide

## 🚀 Quick Start (5 minutes)

### 1. Run Setup Script
```bash
# Save the setup script as setup-daysave.sh
chmod +x setup-daysave.sh
./setup-daysave.sh
```

### 2. Install Additional Dependencies
```bash
cd daysave-app
npm install express express-session express-rate-limit express-validator ejs sequelize mysql2 bcrypt jsonwebtoken passport passport-google-oauth20 passport-microsoft passport-apple helmet cors speakeasy qrcode uuid i18n winston nodemailer multer crypto libphonenumber-js sanitize-html validator dotenv axios sharp compression morgan fingerprintjs2

```

### 3. Copy Required Files

#### A. Database Setup Script
Copy the `setup-db.js` content to `scripts/setup-db.js`

#### B. Image Optimization Script  
Copy the `optimize-images.js` content to `scripts/optimize-images.js`

#### C. Main Application File
Copy the complete `app.js` content (the full version with all routes and middleware)

#### D. EJS Templates
Copy these files to `views/`:
- `index.ejs` (Landing page)
- `login.ejs` (Login page) 
- `register.ejs` (Registration page)
- `dashboard.ejs` (User dashboard)
- `contacts.ejs` (Contacts management)
- `terms.ejs` (Terms of trade)
- `privacy.ejs` (Privacy policy) 
- `contact.ejs` (Contact form)
- `404.ejs` (404 error page)
- `error.ejs` (General error page)

#### E. Localization Files
Copy these files to `locales/`:
- `en.json` (English translations)
- `de.json` (German translations)
- `fr.json`, `it.json`, `es.json` (other languages)

### 4. Configure Environment
```bash
# Edit .env file with your actual values
nano .env
```

Required settings:
```env
NODE_ENV=development
PORT=3000
SESSION_SECRET=your-super-secret-key-here

# Database (MySQL required)
DB_HOST=localhost
DB_NAME=daysave
DB_USER=root
DB_PASSWORD=your-mysql-password

# OAuth (optional for MVP)
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret
```

### 5. Add Your Logo
```bash
# Place your logo file (recommended: 512x512px PNG)
cp /path/to/your/logo.png public/images/daysave-logo.png
```

### 6. Set Up Database
```bash
# Make sure MySQL is running, then:
npm run db:setup
```

### 7. Optimize Images
```bash
npm run logo:optimize
```

### 8. Start Development Server
```bash
npm run dev
```

🎉 **Your DaySave app is now running at http://localhost:3000**

## 🗄️ Database Setup Details

The `setup-db.js` script creates:

### Tables Created:
1. **users** - User accounts and profiles
2. **social_accounts** - Connected social media accounts  
3. **contacts** - Apple iPhone-compatible contacts
4. **contact_groups** - Contact organization
5. **contact_group_members** - Group membership
6. **relationships** - Contact relationships
7. **urls** - Submitted social media URLs
8. **files** - Uploaded files
9. **url_groups** - Content organization
10. **share_logs** - Content sharing history
11. **login_attempts** - Security tracking
12. **contact_submissions** - Contact form messages
13. **admin_settings** - Admin configuration
14. **sessions** - User sessions

### Default Admin Account:
- **Email**: admin@daysave.app
- **Password**: admin123!
- **Note**: Change this in production!
